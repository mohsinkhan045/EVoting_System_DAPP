# Voting System DApp

## Overview

This decentralized voting system DApp is built using ReactJS for the frontend and Tailwind CSS for a sleek and modern design. Here's a step-by-step guide to navigating the platform:

### 1. **Login/Signup:**
   - Clicking the login/signup button seamlessly connects your Metamask wallet, ensuring a secure and user-friendly experience.

### 2. **Token Information:**
   - Navigate to the 'About' section to access essential token details:
     - Token address
     - Token price
     - Remaining tokens
     - Decimal precision

### 3. **Token Purchase and Voting:**
   - Use the 'Buy Token' button to purchase tokens required for voting.
   - Click the 'Vote' button to view token name and symbol, enabling you to make an informed decision.

### 4. **Age Verification:**
   - After token purchase, submit your age. Only users above 18 can proceed, ensuring a responsible voting environment.

### 5. **Candidate Selection and Voting:**
   - Choose your preferred candidate by reading their descriptions.
   - Click 'Vote' to cast your vote securely.

### 6. **Candidate Profiles:**
   - Explore candidate profiles post-voting, displaying:
     - Candidate name
     - Description
     - Optional message

### 7. **Transaction and Candidate Status:**
   - Upon clicking a candidate, view their name, description, and optional message.
   - A transaction occurs, confirming the successful casting of your vote.
   - Candidates appear below with their names, votes, and addresses.

### 8. **Token and Candidate Restrictions:**
   - Token and candidate creation are one-time actions, ensuring fairness.
   - Once a token is bought or a candidate is created, additional purchases or creations are restricted.
   - Only the owner has the privilege of creating candidates and buying tokens.

## Technologies Used:
- **Frontend:** ReactJS
- **Design:** Tailwind CSS

## Usage Instructions:

1. **Clone the Repository:**
   ```
   git clone [repository_url]
   ```

2. **Navigate to Project Directory:**
   ```
   cd [project_directory]
   ```

3. **Run the Application:**
   ```
   npm run dev
   ```

4. **Access the DApp:**
   Open your web browser and go to `http://localhost:5173/`.
